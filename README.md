# ALBERT
Реализация модели ALBERT согласно оригинальной статье на данных WikiText-2

В albert.ipynb содержится основная работа по обработке тренировочных данных и обучению модели с оценкой качества.
В albert.py реализована сама модель ALBERT.
Файлы albert_N.pth являются промежуточными чекпоинтами.

Готово:
- сбор и анализ данных;
- построение модели;
- пайплайн обучения.
